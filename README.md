# AI Recommendation Logic – Tech Stack Recommender

 A content‑based recommendation engine that maps user skills to relevant job roles using TF‑IDF weighting and cosine similarity.

## 📌 Overview

This project is my implementation of **Project 3** from the DecodeLabs Artificial Intelligence curriculum. It marks the shift from passive classification to active prediction — building a system that understands user intent and delivers personalized recommendations without relying on other users' behavior.

The **Tech Stack Recommender** takes a user's raw skills and career interests, then returns the **Top 3 most relevant job roles** based on mathematical similarity scoring.

---

## 🎯 Objective

Build a content‑based filtering engine that:

| Component | Description |
|-----------|-------------|
| **Input** | User provides 3+ skills/interests |
| **Process** | TF‑IDF vectorization + Cosine Similarity scoring |
| **Output** | Top‑3 matching job roles from dataset |
